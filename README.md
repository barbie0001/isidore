# isidore
Инструкция по запуску

Создайте базу данных salon_anora в MySQL.
Импортируйте структуру базы данных.
Отредактируйте файл конфигурации подключения к базе данных db.php, указав правильные параметры.
Настройте веб-сервер (Apache) на корневую директорию проекта (domains).
Убедитесь, что PHP включен и правильно настроен на веб-сервере.
Откройте веб-браузер и перейдите по URL вашего сайта (http://localhost/salon_anora).
